# Modern School Website

A modern, responsive school website built with HTML, CSS, and JavaScript featuring:

## Features

- **3D Hero Section**: Interactive Three.js scene with cursor-responsive animations
- **GSAP Animations**: Smooth scroll-triggered animations and transitions
- **Fully Responsive**: Mobile-first design that works on all devices
- **Modern UI**: Clean, professional design with gradient backgrounds
- **Interactive Elements**: Hover effects, button animations, and form handling

## Technologies Used

- HTML5 (Semantic markup)
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Three.js (3D graphics)
- GSAP (Animations)
- Font Awesome (Icons)
- Google Fonts (Typography)

## Setup Instructions

1. **Download/Clone** the project files
2. **Open** `index.html` in a modern web browser
3. **No build process required** - runs directly in the browser

## File Structure

```
├── index.html          # Main HTML file
├── styles.css          # CSS styles and responsive design
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## Features Breakdown

### 3D Hero Section

- Central icosahedron representing the school
- Floating geometric shapes around the main object
- Mouse movement creates shake/shake effect
- Smooth camera movement following cursor

### Responsive Design

- Mobile-first approach
- Breakpoints: 768px (tablet), 480px (mobile)
- Collapsible navigation menu
- Flexible grid layouts

### Animations

- Scroll-triggered animations using GSAP ScrollTrigger
- Fade-in, slide-in, and scale-in effects
- Counter animations for statistics
- Button hover and click effects

### Sections

- **Hero**: 3D interactive scene with call-to-action buttons
- **About**: School information with animated statistics
- **Programs**: Educational programs with hover effects
- **Facilities**: School facilities with icon cards
- **Contact**: Contact form with validation

## Browser Support

- Chrome (recommended)

## Performance Notes

- Optimized Three.js scene with efficient rendering
- Lazy loading animations
- Responsive images and icons
- Minimal external dependencies

## Customization

To customize the website:

1. **Colors**: Modify CSS custom properties in `styles.css`
2. **Content**: Update text content in `index.html`
3. **3D Scene**: Adjust Three.js parameters in `script.js`
4. **Animations**: Modify GSAP timelines in `script.js`
